# PhyB/PIF Workflow Fit

## Likely Fit

- `standard-construct-loop`: useful when chromophore biosynthesis, localization, and signaling-output geometry need coordinated testing
- `fast-no-cloning-screen`: useful only when an enablement cassette is already in hand and the goal is narrow feasibility testing

## Why

This system has a strong practicality dependency on chromophore handling, so workflow fit is often dominated by build complexity rather than by the dimerizer concept alone.

## Missing Evidence

- comparative turnaround versus blue-light dimerizers
- typical failure points for PCB enablement constructs
- host-specific burden from the support cassette
