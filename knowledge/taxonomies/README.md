# Taxonomy Seeds

This directory is reserved for controlled vocabularies and human-readable taxonomy notes that sit above the raw enum lists in `schemas/canonical/controlled_vocabularies.v1.json`.

Planned seeds:

- mechanism families — biophysical mechanisms of molecular tools (photocleavage, heterodimerization, etc.)
- technique families — engineering methodology categories (computational design, selection/enrichment, etc.)
- validation context hierarchy
- target processes
- workflow families
- citation roles
