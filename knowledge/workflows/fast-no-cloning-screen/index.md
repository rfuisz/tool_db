# Fast No-Cloning Screen

## Objective

Get a fast yes or no signal on a known sequence or small design set with minimal build overhead.

## Recommended For

- quick feasibility screens
- contexts where host toxicity may confound slower cell-based loops
- early triage before committing to a larger construct campaign

## Canonical Step Pattern

1. design candidate
2. order DNA or fragment
3. express or assay in a fast screen format
4. analyze and decide

## Timing Notes

The spec assumes same-day assay after DNA arrival is plausible for the screening step itself; total loop time is still dominated by DNA acquisition lead time.

## Why This Matters

This workflow should stay explicit in the database so "fast to first test" can be computed from steps rather than guessed from item reputation.
